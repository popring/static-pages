## 简介

本项目为一个网页云音乐精选，选出几首自己特别喜欢的歌曲，来做宣传页面。

`gulp` 打包

`sass` css预编译

`swiper` 插件

`jquery`

`animate` 动画

## 运行

```bash

## 安装依赖
npm i

## 打包文件
gulp build

```
